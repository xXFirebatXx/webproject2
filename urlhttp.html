HTTP

Http står för HyperText Transfer Protocol. Det är ett kommunikationsprotokoll som används för att överföra webbsidor på informationsnätverket World Wide Web på Internet.

URL

URL eller Uniform Resource Locator är den teckensträng som identifierar en webbsida som till exempel: www.google.com
